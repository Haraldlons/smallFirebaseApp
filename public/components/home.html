Velkommen til Haralds kule app

Har laget denne appen for å kort vise hvordan man kan bruke Firebase som database.

Firebase har mange fordeler, og gjør kommunikasjonen mellom databasen og clienten mye enklere enn tradisjonelle back-end løsninger som mySQL.

Det er også mye sikrere siden man spesifiserer klare regler for hvem som kan aksessere hva.
